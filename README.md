# 🚀 주요 기능

* 사칙연산(+, -, *, /) 지원
* Enum을 활용한 연산자 관리
* Generic을 활용한 숫자 타입 확장
* 연산 결과 저장
* 연산 결과 조회 및 삭제
* Stream을 활용한 조건별 결과 조회
* 예외 처리(0으로 나누기, 잘못된 연산자 입력)

# 📂 클래스 구조

### App

* 사용자 입력 처리
* 계산 결과 출력
* 프로그램 실행 및 종료 관리

### ArithmeticCalculator

* 사칙연산 수행
* 연산 결과 저장
* 연산 결과 조회 및 삭제

### OperatorType

* 연산자(Enum) 관리

# 🔄 실행 흐름

1. 숫자 2개 입력
2. 연산자 입력
3. 연산 수행
4. 결과 저장 및 출력
5. 조건에 맞는 결과 조회
6. 필요 시 결과 삭제
7. exit 입력 시 종료
